# golf-clap

golf scorecard

```
const Game = require('golf-clap');

const game = new Game({
    // default: 18
    holes: 9,

    players: [
        'Jeff',
        'Susan',
        'Janice',
        'Howard'
    ]
});
```

## Installation

If using `yarn`:
```
yarn add wines
```

Otherwise:
```
npm install wines --save
```
